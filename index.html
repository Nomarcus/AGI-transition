Narrow AI (ANI)

Artificial Narrow Intelligence (ANI) refers to AI systems built to perform a specific task extremely well, but without the ability to operate beyond their domain.

  • Language models: ChatGPT, Claude
  • Image generation: DALL-E, Midjourney
  • Code assistants: GitHub Copilot
  • Medical diagnostics: Image recognition
  • Games: AlphaGo, OpenAI Five

Limitation: These systems excel only within their domain.
